הקדמה

במחברת הקודמת למדתם על CSS באופן כללי –
קיבלתם שיעור היכרות נחמד וקצרצר על CSS והרעיונות הכלליים מאחוריו.

עכשיו, כשאתם יודעים איך נראה התחביר של השפה וצברתם ניסיון בלכתוב דבר או שניים באמצעותה,
הגיע הזמן ללמוד לעומק כל מה שחשוב לדעת כדי לעצב אתר בסיסי.

היום אתם הולכים לחדד את הכישורים שלכם בכתיבת Selector־ים, להבין איך עובד מודל ה־box־ים של CSS ובכללי להשתפשף בקצת פיצ’רים ויזואליים נחמדים של CSS.
כשתסיימו את החומר של היום, תוכלו לגשת בנוחות ליום האחרון, בו תלמדו על רעיונות מתקדמים שעומדים בבסיס של CSS.
