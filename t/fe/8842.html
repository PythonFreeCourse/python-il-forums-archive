היי כולם,

בגלל שאנחנו מקבלים הרבה פניות בנוגע לעיצוב האתר (למשל: מה הליאווט, CSS להשתמש, אם יש דברים להתבסס עליהם, אילו דברים יהיו דינמים, ואיפה הם נפתחים וכו…), ומכיוון שאין באפשרותנו להיות נוכחים בכל הקבוצות שמתקיימות במקביל.

החלטנו לפתוח אשכול בו @Ode ו @sagi.zo ואני נוכל לענות לכם על שאלות, לתת הכוונה בנושאים של UI (user interface) וגם בנושאים של UX במידה ויש.

אני אתחיל שאלה שנשאלנו השבוע דיי הרבה

שאלה: "האם יש css להתבסס עליו?"
תשובה: אנחנו עובדים במקביל עם bootstrap ועם מחלקות css שאנחנו מייצרים בעצמנו שתואמות את העיצוב שאנו שואפים להגיע אליו. לכן אני יכול להציע 2 אפשרויות:

  1. להשתמש בבוסטראם בשביל דברים בסיסיים כמו כפתורים, inputs וכו…
  2. לעשות עם מינימום דיזיין - אבל בצורה שיהיה מובן איך אתם רוצים שהדברים יהיו.

לפה ולכאן, נצטרך כולנו לעשות התאמות לדיזיין אבל זה יהיה שכבר יהיו דפים ויהיה אפשר באמת להתייחס.

אתם מוזמנים תמיד להעלות פה צילומי מסך של דברים שעשיתם, לביקורת (לא רק מאיתנו כמובן מכל חברי הקורס)

לצד הטכני, אנחנו נשתדל להיות פה זמינים לשאלות והתייעצויות נוספות אם יש.
