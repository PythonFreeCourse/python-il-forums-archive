שלום לכולם,

פתחתי פוסט זה כדי לעזור למי שלא הצליח לפתוח את המחברות אוטומטית ב-Chrome בעת כתיבת Jupyter Notebook בשורת הכתובות (File Explorer), כפי שהציג ים בשיעור.

  1. תריצו את הקוד הבא בשורת הכתובות:
    jupyter notebook --generate-config
    הקוד הזה ייצר לכם קובץ קונפיגורציה שבאמצעותו תוכלו לשלוט בהגדרות של שרת הג’ופיטר.

  2. גשו לתיקיית ה-User שלכם (בברירת מחדל נמצאת בשולחן עבודה).
    פתחו את תיקיית .jupyter ושם תמצאו את הקובץ jupyter_notebook_config.
    פתחו אותו באמצעות Notepad.

  3. חפשו בקובץ את השורה:
    c.NotebookApp.browser# = “”
    שנו אותה לשורה:
    c.NotebookApp.browser = u’C:/Program Files (x86)/Google/Chrome/Application/chrome.exe %s’

חשוב להוריד את הסימן # בתחילת השורה!

כעת שמרו את הקובץ.
כאשר תריצו את Jupyter Notebook בשורת הכתובות בתיקיית המחברות שהורדתם, השרת אמור לפתוח את דפדפן Chrome אוטומטית.
